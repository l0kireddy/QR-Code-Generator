# Simple QR Code Generator (HTML + Flask)

## Files
- backend/app.py
- backend/requirements.txt
- frontend/index.html

## Run locally
### Backend
cd backend
pip install -r requirements.txt
python app.py

### Frontend
Open `frontend/index.html` in your browser.

## Deploy
- Backend: Render/Replit
- Frontend: Any static host (or just open the HTML file)
